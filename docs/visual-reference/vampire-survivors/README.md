# Vampire Survivors Visual Acceptance References

These images are local reference material for Infinite Starwish visual QA only.
They are not game assets, style targets, source material for derived art, or
allowed runtime imports.

Source: official Steam app screenshots for Vampire Survivors, app id `1794680`,
retrieved from Steam Store API on 2026-06-06.

Source page:

- https://store.steampowered.com/app/1794680/Vampire_Survivors/
- https://store.steampowered.com/api/appdetails?appids=1794680&filters=screenshots

Use these references to judge:

- Horde density and the feeling of being surrounded.
- Combat readability under many enemies, pickups, and attack effects.
- Projectile and effect saturation without hiding the player.
- Late-run screen energy and reward feedback.
- Whether the first playable visual pass reaches a polished commercial
  spectacle bar instead of looking like programmer art.

Do not use these references for:

- Gothic fantasy theme, character design, UI style, palette, or lore.
- Copying sprites, effects, composition, or image pixels.
- Imported runtime art.
- Training, deriving, tracing, or color-picking runtime assets.

Infinite Starwish remains Neon Pixel Space: a code-drawn, shader-heavy,
space-mech cyberpunk pixel roguelite.

## Acceptance Use

During visual QA, compare local desktop and mobile screenshots against these
images for density, polish, foreground/background separation, player
readability, attack readability, pickup readability, and late-run screen energy.
The comparison is qualitative. It must not become an exact pixel, palette,
composition, or sprite-matching requirement.
