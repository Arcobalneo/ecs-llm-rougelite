# Use DOM UI Around Pixi Combat

Pixi owns the real-time combat canvas, pixel entities, particles, shaders, XP shards, enemies, bosses, and high-frequency game visuals, while DOM and CSS own home, leaderboard, wish input, settlement, language controls, and other system UI. This keeps mobile text input and responsive screens reliable without weakening the code-drawn art rule, since DOM UI is still authored with code rather than image assets.
